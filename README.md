# Chess.Net
A .Net Chess Move Validator library 

## What is done
- Pieces classes
- Pawn and Rook movement validation
- ChessBoard class (composed by 8*8 Case objects)
- Console user input reader for developement testing (move pieces through chessboard)
- Unit tests
